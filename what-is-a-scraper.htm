What is a scraper?

A web scraper is a program that collects data from a website. Scrapers can have a variety of uses, but generally the idea is to get information of some kind from one place and either put it elsewhere or collect it for immediate use. A simple scraper would generally be given a certain URL that it would download information from, then would parse that to extract just the information you want in a specified format. In order for a scraper to be effective, the data its scraping needs to have a consistent format that can be programatically manipulated. If a scraper is trying to gather data from multiple pages for example, malformed HTML tags on the page can cause problems.
